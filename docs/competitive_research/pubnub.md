# PubNub - Detailed Pricing Analysis

## Overview

PubNub is a real-time communication platform that provides pub/sub messaging, presence, and other real-time features.

**Website**: https://www.pubnub.com/
**Pricing Page**: https://www.pubnub.com/pricing/

---

## Pricing Model

PubNub uses a **Monthly Active Users (MAU)** pricing model - the only metric you pay for.

**Key differentiator**: Unlike competitors, PubNub includes unlimited messages, unlimited channels, unlimited connections, and bandwidth in the MAU pricing.

## Pro Plan Pricing Formula

PubNub Pro uses a tiered pricing model with volume discounts:

### Tier 1: Up to 10K MAU
- **Base**: $130/month
- **Per MAU**: $0.04
- **Cap**: $500/month maximum

**Examples**:
- 1,000 MAU = $130 + (1,000 × $0.04) = **$170/month**
- 5,000 MAU = $130 + (5,000 × $0.04) = **$330/month**
- 10,000 MAU = **$500/month** (capped)

### Tier 2: 10,001 to 50,000 MAU
- **Base**: $500/month
- **Per additional MAU**: $0.035

**Examples**:
- 15,000 MAU = $500 + (5,000 × $0.035) = **$675/month**
- 25,000 MAU = $500 + (15,000 × $0.035) = **$1,025/month**
- 50,000 MAU = $500 + (40,000 × $0.035) = **$1,900/month**

### Tier 3: 50,000+ MAU
- **Custom pricing**: Contact sales
- **Volume discounts**: Significant savings at scale
- **Estimated**: ~$3,000-5,000/month for 100K MAU
